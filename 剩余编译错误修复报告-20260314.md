### 1. CS0272 - StoragePath的set访问器不可访问（2个错误）

**位置**:
- SolutionManager.cs:110
- SolutionManager.cs:254

**问题**:
SolutionFile.StoragePath的set访问器被声明为private，但SolutionManager尝试设置它。

**修复**:
```csharp
// 修改前
[JsonIgnore]
public string? StoragePath { get; private set; }

// 修改后
[JsonIgnore]
public string? StoragePath { get; set; }
```

**文件**: `src/Workflow/SolutionFile.cs`

### 2. CS1503 - SaveSolution参数类型不匹配（1个错误）

**位置**:
- ProjectConfigurationDialogViewModel.cs:73

**问题**:
```csharp
_solutionManager.SaveSolution(_selectedSolution);
```
SaveSolution方法期望的是`string? filePath`参数，但传入的是`SolutionFile`对象。

**修复**:
```csharp
// 修改前
_solutionManager.SaveSolution(_selectedSolution);

// 修改后
_solutionManager.SaveSolution(_selectedSolution.StoragePath);
```

**文件**: `src/UI/ViewModels/ProjectConfigurationDialogViewModel.cs`

---

private readonly List<SolutionFile> _solutions;

                                        string? currentDescription, List<SolutionFile> solutions)

#### 3.4 更新ProjectConfigurationDialogViewModel.cs的SaveRecipeAs方法
```csharp
// 修改前
var dialog = new SaveRecipeAsDialog(
    SelectedSolution.Id,
    SelectedRecipe.Name,
    SelectedRecipe.Description,
    Solutions.ToList());

// 修改后 - 添加目标解决方案选择逻辑
var dialog = new SaveRecipeAsDialog(
    SelectedSolution.Id,
    SelectedRecipe.Name,
    SelectedRecipe.Description,
    Solutions.ToList());

if (result == true)
{
    // 获取目标解决方案
    var targetSolution = Solutions.FirstOrDefault(s => s.Id == dialog.TargetSolutionId);
    if (targetSolution == null)
    {
        MessageBox.Show("请选择目标解决方案", "错误", ...);
        return;
    }

    var workflowRef = SelectedRecipe.WorkflowRef;
    var newRecipe = targetSolution.AddRecipe(dialog.RecipeName, workflowRef, dialog.Description);

    // 如果目标不是当前解决方案，需要刷新当前列表
    if (targetSolution.Id != SelectedSolution.Id)
    {
        Recipes.Add(newRecipe);
    }

    SelectedRecipe = newRecipe;
}
```

**文件**: `src/UI/ViewModels/ProjectConfigurationDialogViewModel.cs`
