### 5. 数据加载逻辑

**修改前**:
```csharp
private void LoadData()
{
    LoadWorkspaceFolders();
    LoadRecentProjects();
    LoadProjectsFromWorkspaces();

    if (!string.IsNullOrEmpty(_preselectProjectId))
    {
        var project = Projects.FirstOrDefault(p => p.Id == _preselectProjectId);
        if (project != null)
        {
            SelectedProject = project;
            // ...
        }
    }
}
```

**修改后**:
```csharp
private void LoadData()
{
    LoadWorkspaceFolders();
    LoadRecentSolutions();
    LoadSolutionsFromWorkspaces();

    if (!string.IsNullOrEmpty(_preselectSolutionId))
    {
        var solution = Solutions.FirstOrDefault(s => s.Id == _preselectSolutionId);
        if (solution != null)
        {
            SelectedSolution = solution;
            // ...
        }
    }
}
```

### 6. 配方相关方法

**AddRecipe()**:
- 修改：从 `SelectedProject.AddRecipe()` 改为直接创建 Recipe 对象并添加到 `SelectedSolution`

**DuplicateRecipe()**:
- 修改：从 `SelectedProject.DuplicateRecipe()` 改为 `SelectedRecipe.Clone()`

**SaveRecipeAs()**:
- 修改：添加 TODO 说明需要实现 `SolutionManager.SaveRecipeAs()` 方法
- 临时方案：直接克隆并添加到当前解决方案

**DeleteRecipe()**:
- 修改：从 `SelectedProject.RemoveRecipe()` 改为 `SelectedSolution.RemoveRecipe()`

### 7. 启动逻辑

**修改前**:
```csharp
private void Launch()
{
    if (SelectedProject != null)
    {
        _projectManager.SetCurrentProject(SelectedProject.Id, SelectedRecipe?.Name ?? "standard");
        _launchResult = (SelectedProject, SelectedRecipe);
        LogSuccess($"启动项目: {SelectedProject.Name}");
    }
}
```

**修改后**:
```csharp
private void Launch()
{
    if (SelectedSolution != null)
    {
        _solutionManager.SetCurrentSolution(SelectedSolution.Id, SelectedRecipe?.Name ?? "default");
        _launchResult = (SelectedSolution, SelectedRecipe);
        LogSuccess($"启动解决方案: {SelectedSolution.Name}");
    }
}
```
