## 产品概述

为图像阈值工具实现专业级「结果显示」Tab，提供工业视觉软件标准的结果判断、图像显示控制和文本标注功能。

## 核心功能

### 结果判断分组

- 输出为空判断：检测输出图像是否全黑
- 白色像素比例判断：判断二值化后白色像素占比（范围配置）
- 输出均值判断：判断输出图像平均灰度值（范围配置）
- 输出面积判断：判断白色像素总面积（范围配置）
- 质心X/Y判断：判断白色像素质心坐标（范围配置）

### 图像显示分组

- 输出图像显示控制（眼睛图标切换显示/隐藏）
- 阈值分界线显示控制
- ROI区域显示控制
- 直方图显示控制
- 样式设置弹窗（颜色、透明度、粗细）

### 文本显示分组

- 文本内容配置（支持变量占位符）
- OK/NG颜色选择
- 字号、透明度、位置配置

### 底部操作按钮

- 连续执行、执行、确定按钮

## 技术栈选择

- **框架**: WPF + XAML
- **架构模式**: MVVM
- **基础控件**: 复用 Plugin.SDK 已有控件
- **主题色**: #FF7F00（工业视觉软件标准橙色）

## 实施策略

### 复用现有控件

| 控件 | 来源 | 用途 |
| --- | --- | --- |
| `Expander` | WPF原生 + AppResources样式 | 折叠分组容器 |
| `NumericUpDown` | Plugin.SDK/UI/Controls | 数值输入和范围配置 |
| `CheckBox` | WPF原生 | 开关替代ToggleSwitch |
| `Button` | WPF原生 + PrimaryButton样式 | 图标按钮和操作按钮 |


### 新增文件

| 文件 | 用途 |
| --- | --- |
| `ThresholdResultConfig.cs` | 结果判断配置模型 |
| `ThresholdDisplayConfig.cs` | 图像显示配置模型 |

## 实现细节

### ThresholdResultConfig.cs 结构

```
public class ThresholdResultConfig : ObservableObject
{
    // 结果判断开关
    public bool IsEmptyCheckEnabled { get; set; }
    public bool IsWhitePixelRatioCheckEnabled { get; set; }
    public bool IsMeanCheckEnabled { get; set; }
    public bool IsAreaCheckEnabled { get; set; }
    public bool IsCentroidXCheckEnabled { get; set; }
    public bool IsCentroidYCheckEnabled { get; set; }
    
    // 范围配置
    public double WhitePixelRatioMin { get; set; } = 0;
    public double WhitePixelRatioMax { get; set; } = 100;
    public double MeanMin { get; set; } = 0;
    public double MeanMax { get; set; } = 255;
    // ... 其他范围属性
}
```

### ThresholdDisplayConfig.cs 结构

```
public class ThresholdDisplayConfig : ObservableObject
{
    // 显示项配置
    public DisplayItemConfig OutputImage { get; set; }
    public DisplayItemConfig ThresholdLine { get; set; }
    public DisplayItemConfig Region { get; set; }
    public DisplayItemConfig Histogram { get; set; }
}

public class DisplayItemConfig : ObservableObject
{
    public string Name { get; set; }
    public bool IsVisible { get; set; } = true;
    public Color OkColor { get; set; } = Color.FromRgb(102, 255, 0);
    public Color NgColor { get; set; } = Color.FromRgb(255, 0, 0);
    public double Opacity { get; set; } = 1.0;
    public double Thickness { get; set; } = 1.0;
}
```

## 设计风格

采用工业视觉软件标准设计风格：浅色背景（#F5F5F5）、扁平化专业风格、橙色主题色（#FF7F00）、统一圆角（4px）、规整间距。

## 布局结构

### 结果判断分组

- 标题左对齐，右上角收起/展开按钮
- 每项高度36px，标签左对齐，开关右对齐
- CheckBox替代ToggleSwitch，开启状态橙色高亮
- 开关开启时展开范围配置行，使用NumericUpDown

### 图像显示分组

- 垂直列表布局，每项高度36px
- 左侧显示项名称，中间眼睛图标按钮，右侧铅笔图标按钮
- 图标默认灰色（#999999），激活橙色（#FF7F00）

### 文本显示分组

- 表单式布局，标签右对齐，宽度100px
- 输入框高度32px，圆角4px，间距8px
