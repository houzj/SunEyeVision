用户手册

SunEyeVision 使用指南

快速开始

系统要求

  • 操作系统: Windows 10/11 (64位)
  • .NET Runtime: .NET 9.0 或更高版本
  • 内存: 4GB RAM (推荐 8GB)
  • 硬盘: 500MB 可用空间
  • 显示器: 1920x1080 或更高分辨率

安装步骤

  1. 下载 SunEyeVision 安装包
  2. 双击安装程序，按照提示完成安装
  3. 启动 SunEyeVision 应用程序
  4. 首次启动会初始化配置文件

界面介绍

主窗口布局

SunEyeVision 主窗口包含以下区域：

  • 菜单栏：文件、编辑、视图、工具、帮助
  • 工具栏：常用操作快捷按钮
  • 工作区：工作流设计器
  • 属性面板：编辑算法参数
  • 图像显示区：显示原始图像和处理结果
  • 状态栏：显示当前状态和提示信息

菜单说明

  • 文件：新建、打开、保存、导入、导出
  • 编辑：撤销、重做、复制、粘贴、删除
  • 视图：工作流、图像显示、日志输出
  • 工具：设备管理、插件管理、配置设置
  • 帮助：帮助文档、关于

设备管理

添加设备

  1. 点击菜单 工具 → 设备管理
  2. 点击 添加设备 按钮
  3. 选择设备类型 (相机、光源、I/O)
  4. 配置设备参数 (IP 地址、端口等)
  5. 点击 连接 测试设备
  6. 点击 确定 保存设备配置

配置相机

  • 曝光时间：控制相机曝光时长
  • 增益：控制图像亮度
  • 触发模式：内触发 / 外触发
  • 图像分辨率：设置采集图像大小

配置光源

  • 亮度：调节光源亮度 (0-100%)
  • 频闪模式：常亮 / 频闪
  • 通道：选择光源通道

工作流设计

创建工作流

  1. 点击菜单 文件 → 新建工作流
  2. 在左侧工具箱中选择算法节点
  3. 拖拽节点到工作区
  4. 连接节点形成处理流程
  5. 设置每个节点的参数
  6. 保存工作流

添加算法节点

  1. 在左侧工具箱中找到需要的算法
  2. 拖拽算法节点到工作区
  3. 在属性面板中设置节点参数
  4. 给节点命名便于识别

连接节点

  1. 点击前一个节点的输出端口
  2. 拖拽连接线到下一个节点的输入端口
  3. 释放鼠标完成连接
  4. 可以右键连接线选择删除

设置参数

  1. 选中工作区中的节点
  2. 在右侧属性面板中显示节点参数
  3. 修改参数值
  4. 点击 应用 保存参数

执行工作流

运行工作流

  1. 确保设备已连接并正常工作
  2. 点击工具栏的 运行 按钮
  3. 或者按 F5 快捷键
  4. 观察执行进度和结果

调试工作流

  1. 点击 调试 按钮
  2. 工作流会逐节点执行
  3. 可以在每个节点暂停查看结果
  4. 使用 单步执行 按钮逐步调试
  5. 点击 继续 完成执行

查看结果

  • 图像结果：在图像显示区查看处理后的图像
  • 数据结果：在结果面板查看算法输出数据
  • 执行日志：在日志窗口查看执行记录
  • 性能统计：查看每个节点的执行时间

插件管理

安装插件

  1. 点击菜单 工具 → 插件管理
  2. 点击 安装插件 按钮
  3. 选择插件 DLL 文件
  4. 确认插件信息
  5. 插件自动加载到系统

启用/禁用插件

  1. 在插件列表中找到目标插件
  2. 勾选/取消勾选 启用 复选框
  3. 插件状态立即生效

卸载插件

  1. 在插件列表中选中要卸载的插件
  2. 点击 卸载 按钮
  3. 确认卸载操作
  4. 插件从系统中移除

常用快捷键

快捷键 功能
Ctrl + N 新建工作流
Ctrl + O 打开工作流
Ctrl + S 保存工作流
Ctrl + Z 撤销
Ctrl + Y 重做
Delete 删除选中节点
F5 运行工作流
F10 单步执行
F1 打开帮助

常见问题

Q: 无法连接相机？

A: 请检查以下几点：

  • 相机电源是否打开
  • 网线/USB线是否连接正常
  • 相机IP地址是否在同一网段
  • 相机驱动是否正确安装

Q: 工作流执行报错？

A: 请检查：

  • 节点连接是否正确
  • 参数设置是否合理
  • 查看日志窗口的详细错误信息
  • 尝试单步执行定位问题节点

Q: 图像显示异常？

A: 可能的原因：

  • 图像格式不支持
  • 内存不足
  • 图像尺寸过大
  • 检查图像预览设置

Q: 性能优化建议？

A: 优化建议：

  • 减少不必要的算法节点
  • 使用并行处理功能
  • 适当降低图像分辨率
  • 关闭不必要的实时预览
