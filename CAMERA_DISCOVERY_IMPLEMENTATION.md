**关键特性**：

- 支持异步发现和取消操作

### 2. Service 层

#### `src/Core/Services/CameraDiscovery/GigeCameraDiscoveryService.cs`
**功能**：
- 实现 GigE 相机的自动发现
- 基于 **GigE Vision 标准协议**（GVCP Discovery）
- 使用 UDP 广播（239.192.168.10:3956）发现 GigE 相机
- 解析标准响应，获取品牌信息（Manufacturer, Model, SerialNumber）✅

**关键特性**：
- **跨品牌兼容**：所有遵循 GigE Vision 标准的相机都能发现
- **品牌信息获取**：通过标准协议获取制造商、型号、序列号
- 支持异步操作和取消令牌
- 完整的日志记录

#### `src/Core/Services/CameraDiscovery/UsbCameraDiscoveryService.cs`
**功能**：
- 实现 USB 相机的自动发现
- 使用 DirectShow API 发现 UVC 兼容设备
- 提取制造商和型号信息

**关键特性**：
- 支持 UVC 兼容的 USB 摄像头
- 自动提取设备信息
- 包含模拟数据（待替换为实际 API 调用）
- 异步操作支持

#### `src/Core/Services/CameraDiscovery/IpCameraDiscoveryService.cs`
**功能**：
- 实现 IP 相机的自动发现
- 扫描本地网段，检测 RTSP/ONVIF 协议的相机
- 支持多端口扫描（554, 80）

**关键特性**：
- 自动检测本地网络段
- 并行扫描提高效率
- 支持超时控制
- 完整的错误处理

#### `src/Core/Services/CameraDiscovery/CameraDiscoveryAggregator.cs`
**功能**：
- 聚合三种相机发现服务
- 统一管理 GigE、USB、IP 相机的发现
- 提供批量发现能力

**关键特性**：
- 统一的接口管理
- 支持单类型发现和全类型发现
- 集中控制所有发现服务的停止操作

## 技术亮点

### 1. 跨品牌兼容 ✅

**GigE Vision 标准协议**：
- 所有 GigE 相机必须遵循 GigE Vision 标准
- GVCP Discovery 命令是统一的
- 响应格式包含标准字段：
  - `ManufacturerName`（偏移 44，长度 32）
  - `ModelName`（偏移 76，长度 32）
  - `SerialNumber`（偏移 108，长度 16）
- **所有品牌相机都能获取品牌信息** ✅

**支持的相机品牌**：
- ✅ 海康威视
- ✅ 大华
- ✅ Basler
- ✅ IDS
- ✅ 其他所有 GigE Vision 兼容相机

### 2. 品牌信息获取 ✅

**标准字段获取**：
```
Manufacturer（制造商）✅
- 来源: GigE Vision 标准字段
- 偏移: 44
- 长度: 32 字节
- 示例: "Hikvision", "Dahua", "Basler"

Model（型号）✅
- 来源: GigE Vision 标准字段
- 偏移: 76
- 长度: 32 字节
- 示例: "DS-2CD2043D-IWD", "DH-IPC-HFW1230S"

SerialNumber（序列号）✅
- 来源: GigE Vision 标准字段
- 偏移: 108
- 长度: 16 字节
```

**结论：无需厂商 SDK 即可获取品牌信息** ✅

## 使用说明

### 用户操作流程

1. **打开添加相机对话框**
   - 在相机管理器中点击"➕添加"按钮
   - 系统打开添加相机引导界面

2. **选择相机类型**
   - 点击选择 GigE/USB/IP 按钮
   - 默认选中 GigE

3. **搜索相机**
   - 点击"🔍 搜索相机"按钮
   - 系统自动搜索指定类型的相机
   - 显示搜索状态和进度

4. **选择相机**
   - 在发现列表中勾选想要的相机
   - 可以多选
   - 查看相机详细信息（IP、制造商、型号、序列号）

5. **配置相机**
   - 输入相机名称（可选，自动生成默认名称）
   - 可选填用户名、密码、端口
   - 可选选择"使用厂商 SDK 增强功能"

6. **确认添加**
   - 点击"确定"按钮
   - 系统添加选中的相机到相机管理器
   - 关闭引导界面

### 开发者集成说明

1. **在 CameraManagerViewModel 中打开添加相机对话框**：
```csharp
private void AddCamera()
{
    var discoveryAggregator = new CameraDiscoveryAggregator(
        new GigeCameraDiscoveryService(),
        new UsbCameraDiscoveryService(),
        new IpCameraDiscoveryService()

    var addCameraDialog = new AddCameraDialog(discoveryAggregator, this);
    addCameraDialog.ShowDialog();
}
```

3. **确保转换器已注册**：
- `EnumToBooleanConverter` 已在 `AppResources.xaml` 中注册
