## 文档生成

### 手动生成
```powershell
cd 项目根目录
powershell -ExecutionPolicy Bypass -File tools\GenerateHelpDocumentation.ps1
```

### 跳过构建快速生成
```powershell
powershell -ExecutionPolicy Bypass -File tools\GenerateHelpDocumentation.ps1 -SkipBuild
```

## 文档位置

- **源文件**: `Help/Source/zh-CN/`
- **输出目录**: `Help/Output/`
- **API 文档**: `Help/Output/api/`

## 访问方式

1. **在应用中**：按 `F1` 键或点击菜单 "帮助 → 帮助文档"
2. **在浏览器中**：打开 `Help/Output/index.html`
3. **CHM 格式**：如果安装了 HTML Help Workshop，可以生成 CHM 文件
