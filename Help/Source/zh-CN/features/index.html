核心功能特性

SunEyeVision 强大的功能模块

1. 工作流流程控制

支持复杂的工作流控制结构，让检测流程更灵活、更强大。

循环节点

  • 固定次数循环：按指定次数重复执行节点
  • 条件循环：根据条件判断是否继续循环
  • 集合迭代：对一组数据逐个处理

分支节点

  • 条件分支：根据条件选择不同的执行路径
  • 多分支路由：支持多个分支条件
  • 默认分支：当所有条件都不匹配时执行

并行分支节点

  • 多分支并行执行：同时执行多个分支
  • 并行度控制：限制同时运行的分支数量
  • 结果汇聚：等待所有分支完成后继续

子工作流节点

  • 工作流嵌套：在一个工作流中调用另一个工作流
  • 参数传递：支持输入输出参数
  • 结果映射：将子工作流结果映射到父工作流

异常处理节点

  • Try-Catch-Finally：完整的异常处理结构
  • 异常捕获：捕获并处理执行错误
  • 错误恢复：从错误中恢复或提供回退方案

应用场景

批量检测多个产品、根据检测结果分拣、多相机并行采集、复杂的多步骤检测流程等。

2. 智能节点连线方案

采用先进的路径规划算法，提供智能、美观的节点连线功能。

A* 路径规划算法

  • 智能寻路：自动寻找最优连接路径
  • 网格化画布：将画布划分为网格进行路径搜索
  • 障碍物识别：自动识别节点区域作为障碍

智能避障

  • 碰撞检测：使用 Bresenham 算法检测线段碰撞
  • 智能绕行：自动生成绕过障碍物的路径
  • 路径平滑：减少不必要的拐点，路径更美观

多种连线样式

  • 直线：最简单的直接连接
  • 折线：智能选择拐点，路径清晰
  • 贝塞尔曲线：平滑美观的曲线连接
  • 正交线：水平/垂直的专业外观

路径优化策略

  • 路径缓存：缓存计算结果，提升性能
  • 动态更新：节点移动时自动调整连线
  • 批量更新：同时更新多条连线，避免闪烁

应用场景

复杂的工作流设计、大量节点连接、需要清晰美观的流程图展示等。

3. 灵活的节点执行控制

提供多种执行模式，优化工作流执行效率。

执行模式

  • 同步执行：按拓扑顺序依次执行节点
  • 异步执行：利用多线程并行执行节点
  • 混合模式：关键路径同步，非关键路径异步

依赖管理

  • 依赖图分析：自动分析节点间的依赖关系
  • 拓扑排序：使用 Kahn 算法确定执行顺序
  • 循环依赖检测：检测并防止死循环

执行控制

  • 并行度控制：限制同时执行的节点数量
  • 资源限制：控制内存、CPU 等资源使用
  • 超时控制：设置节点执行超时时间
  • 取消执行：支持中断正在运行的工作流

执行监控

  • 执行历史：记录节点执行历史
  • 性能统计：统计每个节点的执行时间
  • 资源监控：实时监控内存、CPU 使用情况

应用场景

需要高效率的实时检测、大规模并行处理、资源受限的环境等。

4. 强大的脚本编辑器

集成了专业级的代码编辑器，支持自定义算法脚本。

编辑器特性

  • 语法高亮：清晰的代码语法着色
  • 智能提示：自动补全代码和提示 API
  • 语法检查：实时检查代码语法错误
  • 代码格式化：自动格式化代码，保持整洁

调试功能

  • 断点设置：在关键位置设置断点
  • 单步执行：逐行调试代码
  • 变量监视：实时查看变量值
  • 调用栈：查看函数调用链

多语言支持

  • C# 脚本：完整的 C# 语言支持
  • Python 脚本：支持 Python 脚本开发
  • 表达式计算：支持复杂的表达式求值

集成功能

  • 工作流集成：脚本节点可直接调用
  • 算法访问：可调用所有内置算法
  • 设备访问：支持控制和读取设备数据

应用场景

自定义特殊算法、实现复杂的数据处理逻辑、集成第三方库等。

5. 定制化客户运行界面

提供灵活的界面定制功能，满足不同客户的需求。

基础定制

  • Logo 替换：使用客户自己的品牌 Logo
  • 主题配色：自定义软件配色方案
  • 标题文本：修改应用名称和标题

布局定制

  • 面板布局：自定义面板的位置和大小
  • 工具栏配置：显示/隐藏工具栏按钮
  • 菜单定制：自定义菜单项和快捷键

功能定制

  • 权限控制：基于角色的权限管理
  • 功能开关：启用/禁用特定功能模块
  • 操作流程定制：简化操作流程，适合最终用户

工作流模板

  • 预设模板：提供常用检测流程模板
  • 模板锁定：锁定模板，防止误修改
  • 一键加载：快速加载预设流程

配置管理

  • 可视化配置：图形化配置界面
  • 配置导入导出：方便在不同环境间迁移
  • 多配置支持：支持多套配置方案切换

用户和角色管理

  • 用户管理：创建和管理系统用户
  • 角色定义：定义不同角色和权限
  • 操作日志：记录用户操作历史

应用场景

OEM 合作、为不同客户定制专用界面、简化操作以适合操作员使用等。
