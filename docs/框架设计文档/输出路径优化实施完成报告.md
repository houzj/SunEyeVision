## 关键配置说明

### Directory.Build.props 配置

```xml
<!-- Debug 输出路径 -->
<OutDir Condition="'$(Configuration)'=='Debug'">$(MSBuildProjectDirectory)\..\dev\Debug\</OutDir>

<!-- Release 输出路径 -->
<OutDir Condition="'$(Configuration)'=='Release'">$(MSBuildProjectDirectory)\..\Release\SunEyeVision_v1.0.0\SunEyeVision\</OutDir>

<!-- 工具插件特殊配置 -->
<PropertyGroup Condition="'$(IsToolPlugin)'=='true'">
  <OutDir Condition="'$(Configuration)'=='Debug'">$(MSBuildProjectDirectory)\..\dev\Debug\plugins\</OutDir>
  <OutDir Condition="'$(Configuration)'=='Release'">$(MSBuildProjectDirectory)\..\Release\SunEyeVision_v1.0.0\SunEyeVision\plugins\</OutDir>
</PropertyGroup>

<!-- UI项目特殊配置 -->
<PropertyGroup Condition="'$(OutputType)'=='WinExe'">
  <OutDir Condition="'$(Configuration)'=='Debug'">$(MSBuildProjectDirectory)\..\dev\Debug\</OutDir>
  <OutDir Condition="'$(Configuration)'=='Release'">$(MSBuildProjectDirectory)\..\Release\SunEyeVision_v1.0.0\SunEyeVision\</OutDir>
</PropertyGroup>

<!-- 中间输出目录 -->
<IntermediateOutputPath>$(MSBuildProjectDirectory)\..\build_temp\obj\$(MSBuildProjectName)\$(Configuration)\$(TargetFramework)\</IntermediateOutputPath>
```

### 插件路径配置

## 使用方式

### Debug 模式开发
1. 在 Visual Studio 中打开 `SunEyeVision.sln`
2. 选择 **Debug** 配置
3. 点击 **生成 → 生成解决方案** (或按 F6)
4. 输出路径: `dev/Debug/`
5. 运行主程序: `dev/Debug/SunEyeVision.exe`

### Release 模式发布
1. 在 Visual Studio 中打开 `SunEyeVision.sln`
2. 选择 **Release** 配置
3. 点击 **生成 → 生成解决方案** (或按 F6)
4. 输出路径: `Release/SunEyeVision_v1.0.0/SunEyeVision/`
5. 添加发布文档:
   - `Release/SunEyeVision_v1.0.0/README.txt`
   - `Release/SunEyeVision_v1.0.0/License.txt`
   - `Release/SunEyeVision_v1.0.0/Install.txt`
6. 打包 `Release/SunEyeVision_v1.0.0/` 目录为压缩包或安装程序

### 清理构建输出
- 在 Visual Studio 中: **生成 → 清理解决方案**
- 或手动删除 `dev/` 和 `Release/` 目录
