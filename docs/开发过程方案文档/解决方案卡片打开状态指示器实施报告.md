## 实施内容

### 1. ViewModel 更新 (`SolutionConfigurationDialogViewModel.cs`)

#### 新增属性
```csharp
// 辅助属性：当前打开的解决方案ID（用于XAML数据触发器）
public string? CurrentSolutionId => _solutionManager.CurrentSolution?.Id
                                   ?? _solutionManager.Settings.CurrentSolutionId;
```

#### 属性通知更新
```csharp
public Solution? SelectedSolution
{
    get => _selectedSolution;
    private set
    {
        if (SetProperty(ref _selectedSolution, value))
        {
            OnPropertyChanged(nameof(SelectedSolutionId));
            OnPropertyChanged(nameof(CurrentSolutionId));  // ✅ 新增通知
        }
    }
}
```

### 2. XAML 更新 (`SolutionConfigurationDialog.xaml`)

#### 右下角角标实现
位置：卡片内容容器的 Grid 内部，主内容区域之前

```xml
<!-- 右下角"打开"状态角标（方案1+方案4组合） -->
<Border HorizontalAlignment="Right" VerticalAlignment="Bottom"
        Margin="0,0,-4,-4"
        Width="52" Height="20"
        CornerRadius="10"
        Background="#2196F3"
        BorderBrush="White"
        BorderThickness="2">
    <Border.Style>
        <Style TargetType="Border">
            <Setter Property="Visibility" Value="Collapsed"/>
            <Setter Property="Effect">
                <Setter.Value>
                    <DropShadowEffect Color="#000000" BlurRadius="4" ShadowDepth="2" Opacity="0.3"/>
                </Setter.Value>
            </Setter>
            <Style.Triggers>
                <!-- 当前打开的解决方案 -->
                <DataTrigger Value="True">
                    <DataTrigger.Binding>
                        <MultiBinding Converter="{StaticResource EqualityConverter}">
                            <Binding Path="Id"/>
                            <Binding Path="DataContext.CurrentSolutionId" RelativeSource="{RelativeSource AncestorType=Window}"/>
                        </MultiBinding>
                    </DataTrigger.Binding>
                    <Setter Property="Visibility" Value="Visible"/>
                </DataTrigger>
            </Style.Triggers>
        </Style>
    </Border.Style>
    <Grid>
        <StackPanel Orientation="Horizontal" HorizontalAlignment="Center" VerticalAlignment="Center">
            <TextBlock Text="📂" FontSize="9" Margin="3,0,1,0"/>
            <TextBlock Text="打开" FontSize="9" Foreground="White" FontWeight="SemiBold" VerticalAlignment="Center"/>
        </StackPanel>
    </Grid>
</Border>
```

## 设计特点

### 视觉设计
1. **位置**：右下角（避免与右上角编辑按钮冲突）
2. **形状**：椭圆形徽章（52x20，圆角半径10）
3. **边距**：`Margin="0,0,-4,-4"` 使徽章部分突出边界，形成徽章效果
4. **颜色**：
   - 背景：蓝色 (#2196F3)
   - 边框：白色（BorderThickness=2）
   - 文字：白色
5. **阴影**：轻微阴影效果，增强立体感

### 内容设计
1. **图标**：📂 文件夹图标（FontSize=9）
2. **文字**："打开"（FontSize=9, FontWeight=SemiBold）
3. **布局**：水平排列，图标在前，文字在后

### 交互设计
1. **触发条件**：当 `Id == DataContext.CurrentSolutionId` 时显示
2. **默认状态**：Collapsed（折叠，不占空间）
3. **数据源**：从 SolutionManager.CurrentSolution 或 SolutionSettings.CurrentSolutionId 获取

## 技术实现

### 数据绑定流程
1. 用户双击启动解决方案
2. `Launch` 方法加载完整 Solution 对象
3. 调用 `SolutionManager.SetCurrentSolution(fullSolution)`
4. 设置 `SolutionManager.Settings.CurrentSolutionId`
5. `SelectedSolution` setter 触发 `OnPropertyChanged(nameof(CurrentSolutionId))`
6. XAML DataTrigger 检测到 `CurrentSolutionId` 变化
7. 右下角角标变为 Visible

### EqualityConverter
使用已存在的 EqualityConverter 比较两个 ID 是否相等：
```xml
<MultiBinding Converter="{StaticResource EqualityConverter}">
    <Binding Path="Id"/>
    <Binding Path="DataContext.CurrentSolutionId" RelativeSource="{RelativeSource AncestorType=Window}"/>
</MultiBinding>
```
