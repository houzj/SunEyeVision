# 区域管理参数优化设计方案

## 📋 规则元数据
- **规则ID**: optimization-001
- **优先级**: 🔴 Critical
- **适用范围**: 区域编辑器参数系统
- **生效模式**: 总是应用
- **最后更新**: 2026-04-13

## 🎯 方案概述

本方案旨在优化区域管理参数系统，统一绘图模式和订阅模式，消除代码重复，提高可维护性。通过引入统一的 `Region` 参数类，替代现有的 `ShapeParameters` 和 `ComputedRegion`，实现两种模式的完全统一。

**核心原则**：
- ✅ 遵循 rule-008：不考虑向后兼容，保持代码干净
- ✅ 遵循 rule-002：命名符合视觉软件风格
- ✅ 遵循 rule-010：使用 `[JsonDerivedType]` 支持多态序列化
- ✅ 使用 `ParamValue<T> + ToolParameters` 模式，复用工具层设计

## 🔍 背景和问题

### 当前问题

1. **数据模型不统一**
   - 绘图模式使用 `ShapeParameters`（继承 `ToolParameters`）
   - 订阅模式使用 `ComputedRegion`（继承 `RegionDefinition`）
   - 两者功能重复，需要维护两套代码

2. **访问模式不一致**
   - 绘图模式：直接访问 `Parameters.CenterX`（常量值）
   - 订阅模式：通过 `RegionResolver` 解析后访问 `ResolvedRegion.CenterX`（解析后的值）
   - 算法层需要区分两种模式，增加复杂度

3. **ViewModel 层臃肿**
   - 存在 `ParameterPanelViewModel` 和 `SubscribePanelViewModel` 两个中间层
   - `RegionEditorViewModel` 需要分别维护两套属性
   - 代码重复，难以维护

4. **命名不符合视觉软件风格**
   - `ComputedRegion` 包含技术术语，不够直观
   - `RegionDefinition` 功能过于简单，仅包含 Id/Name

### 设计目标

1. **统一数据模型**：绘图模式和订阅模式使用同一个 `Region` 对象
2. **统一访问模式**：两种模式都通过 `ParamValue<T>.Value` 访问值
3. **简化架构**：删除中间层，减少代码量
4. **符合命名规范**：使用直观的名称（`Region`）而非技术术语

## 💡 核心设计思想

### 1. ParamValue<T> + ToolParameters 模式

复用工具层的 `ParamValue<T>` 设计，支持常量和动态绑定两种模式：

```csharp
public class ParamValue<T> : ObservableObject
{
    public T Value { get; }           // 解析后的值（常量或绑定）
    public BindingConfig? BindingConfig { get; set; }  // 绑定配置（订阅模式用）
}

// 使用示例
public class Region : ToolParameters
{
    public ParamValue<double> CenterX { get; set; } = ParamValue<double>.CreateConstant(0, "CenterX");
    public ParamValue<double> CenterY { get; set; } = ParamValue<double>.CreateConstant(0, "CenterY");
    
    [JsonIgnore]
    public bool HasAnyBinding => CenterX.BindingConfig != null || CenterY.BindingConfig != null;
}
```

**优势**：
- ✅ 绘图模式：直接使用 `.Value`，无需解析
- ✅ 订阅模式：配置 `.BindingConfig`，自动解析 `.Value`
- ✅ 算法层统一访问模式，无需区分

### 2. 统一的 Region 对象

```csharp
[JsonDerivedType(typeof(Region), "Region")]
public class Region : ToolParameters
{
    public string Id { get; set; } = Guid.NewGuid().ToString();
    public string Name { get; set; } = "新建区域";
    public ShapeType ShapeType { get; set; } = ShapeType.Rectangle;
    
    // 所有形状参数
    public ParamValue<double> CenterX { get; set; } = ParamValue<double>.CreateConstant(0, "CenterX");
    public ParamValue<double> CenterY { get; set; } = ParamValue<double>.CreateConstant(0, "CenterY");
    public ParamValue<double> Width { get; set; } = ParamValue<double>.CreateConstant(100, "Width");
    public ParamValue<double> Height { get; set; } = ParamValue<double>.CreateConstant(100, "Height");
    public ParamValue<double> Angle { get; set; } = ParamValue<double>.CreateConstant(0, "Angle");
    public ParamValue<double> Radius { get; set; } = ParamValue<double>.CreateConstant(50, "Radius");
    public ParamValue<double> StartAngle { get; set; } = ParamValue<double>.CreateConstant(0, "StartAngle");
    public ParamValue<double> EndAngle { get; set; } = ParamValue<double>.CreateConstant(360, "EndAngle");
    
    // 矩形和旋转矩形参数
    public ParamValue<double> Top { get; set; } = ParamValue<double>.CreateConstant(0, "Top");
    public ParamValue<double> Left { get; set; } = ParamValue<double>.CreateConstant(0, "Left");
    public ParamValue<double> Bottom { get; set; } = ParamValue<double>.CreateConstant(100, "Bottom");
    public ParamValue<double> Right { get; set; } = ParamValue<double>.CreateConstant(100, "Right");
    
    // 圆形参数
    public ParamValue<double> InnerRadius { get; set; } = ParamValue<double>.CreateConstant(0, "InnerRadius");
    
    // 辅助属性
    [JsonIgnore]
    public bool HasAnyBinding => CheckAnyBinding();
    
    private bool CheckAnyBinding()
    {
        return CenterX.BindingConfig != null || CenterY.BindingConfig != null ||
               Width.BindingConfig != null || Height.BindingConfig != null ||
               Angle.BindingConfig != null || Radius.BindingConfig != null ||
               StartAngle.BindingConfig != null || EndAngle.BindingConfig != null ||
               Top.BindingConfig != null || Left.BindingConfig != null ||
               Bottom.BindingConfig != null || Right.BindingConfig != null ||
               InnerRadius.BindingConfig != null;
    }
}
```

### 3. 简化的访问模式

**绘图模式**：
```csharp
// UI 绑定
<TextBox Text="{Binding Region.CenterX.Value, UpdateSourceTrigger=PropertyChanged}" />

// 算法访问
double centerX = region.Region.CenterX.Value;
```

**订阅模式**：
```csharp
// UI 绑定配置
<ComboBox SelectedItem="{Binding Region.Angle.BindingConfig, ...}" />

// 算法访问（自动解析）
double angle = region.Region.Angle.Value;  // 自动解析绑定或返回常量
```

**文件**: `src/Plugin.SDK/Execution/Parameters/Region.cs`

```csharp
using SunEyeVision.Plugin.SDK.Execution.Parameters;
using SunEyeVision.Plugin.SDK.Models;
using System.Text.Json.Serialization;

namespace SunEyeVision.Plugin.SDK.Execution.Parameters
{
    /// <summary>
    /// 统一的区域参数类
    /// </summary>
    /// <remarks>
    /// 设计说明：
    /// 1. 使用 ParamValue<T> 支持常量和动态绑定两种模式
    /// 2. 绘图模式：直接使用 .Value 访问常量值
    /// 3. 订阅模式：配置 .BindingConfig，自动解析 .Value
    /// 4. 算法层统一访问模式，无需区分绘图/订阅模式
    /// </remarks>
    [JsonDerivedType(typeof(Region), "Region")]
    public class Region : ToolParameters
    {
        public string Id { get; set; } = Guid.NewGuid().ToString();
        public string Name { get; set; } = "新建区域";
        public ShapeType ShapeType { get; set; } = ShapeType.Rectangle;
        
        // 中心点和尺寸
        public ParamValue<double> CenterX { get; set; } = ParamValue<double>.CreateConstant(0, "CenterX");
        public ParamValue<double> CenterY { get; set; } = ParamValue<double>.CreateConstant(0, "CenterY");
        public ParamValue<double> Width { get; set; } = ParamValue<double>.CreateConstant(100, "Width");
        public ParamValue<double> Height { get; set; } = ParamValue<double>.CreateConstant(100, "Height");
        
        // 旋转角度
        public ParamValue<double> Angle { get; set; } = ParamValue<double>.CreateConstant(0, "Angle");
        
        // 圆形参数
        public ParamValue<double> Radius { get; set; } = ParamValue<double>.CreateConstant(50, "Radius");
        public ParamValue<double> StartAngle { get; set; } = ParamValue<double>.CreateConstant(0, "StartAngle");
        public ParamValue<double> EndAngle { get; set; } = ParamValue<double>.CreateConstant(360, "EndAngle");
        
        // 矩形参数
        public ParamValue<double> Top { get; set; } = ParamValue<double>.CreateConstant(0, "Top");
        public ParamValue<double> Left { get; set; } = ParamValue<double>.CreateConstant(0, "Left");
        public ParamValue<double> Bottom { get; set; } = ParamValue<double>.CreateConstant(100, "Bottom");
        public ParamValue<double> Right { get; set; } = ParamValue<double>.CreateConstant(100, "Right");

                   Angle.BindingConfig != null || Radius.BindingConfig != null ||
                   StartAngle.BindingConfig != null || EndAngle.BindingConfig != null ||
```

## 💡 最佳实践

### 1. 参数类设计

- 使用 `ParamValue<T>` 而非直接使用 `T` 类型
- 提供默认值，确保对象始终处于有效状态
- 使用 `[JsonIgnore]` 标注计算属性

### 2. ViewModel 设计

- 避免中间层，直接暴露数据模型
- 使用数据绑定而非手动属性传递
- 保持 ViewModel 的简洁性

### 3. UI 绑定

- 使用 `UpdateSourceTrigger=PropertyChanged` 实现实时更新
- 使用 `Mode=TwoWay` 支持双向绑定
- 绑定路径简洁明了
